# 局域网访问说明

## 📱 手机访问步骤

### 前提条件
✅ 电脑和手机连接**同一个 WiFi 网络**

### 访问地址

在手机浏览器输入:
```
http://192.168.1.141:8766
```

## ⚠️ 重要说明

### 局域网访问的限制

1. **PWA 无法安装**
   - PWA 必须使用 HTTPS
   - 局域网是 HTTP 协议
   - 无法"添加到主屏幕"

2. **仅限同一 WiFi**
   - 只能连接到同一个路由器
   - 4G/5G 无法访问
   - 不在同一网络无法访问

3. **电脑必须开机**
   - 依赖本地服务器
   - 关闭电脑后无法访问

## 🔧 如果无法访问

### 方法 1: 检查防火墙 (需要管理员权限)

以管理员身份打开 PowerShell,执行:
```powershell
netsh advfirewall firewall add rule name="EventRecruit" dir=in action=allow protocol=TCP localport=8766
```

### 方法 2: 关闭防火墙 (临时,不推荐)

1. 打开 "Windows Defender 防火墙"
2. 点击"启用或关闭 Windows Defender 防火墙"
3. 私有网络设置选择"关闭"
4. 公共网络设置选择"关闭"

**注意**: 完成测试后记得重新开启防火墙!

### 方法 3: 使用其他端口

如果 8766 端口不通,尝试其他端口:

1. 修改 `server.js`:
   ```javascript
   const PORT = 3000; // 改为 3000 或其他端口
   ```

2. 重启服务器:
   ```bash
   node server.js
   ```

3. 访问新地址:
   ```
   http://192.168.1.141:3000
   ```
