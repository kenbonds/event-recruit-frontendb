## 步骤 1: 创建 GitHub Token

由于 GitHub 不再支持密码登录,需要创建 Personal Access Token:

1. 访问 https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 填写信息:
   - **Note**: EventRecruit Deploy
   - **Expiration**: 30 days (或 No expiration)
   - **Scopes**: 勾选 **repo** (完整权限)
4. 点击 "Generate token"
5. **立即复制 Token** (关闭页面后无法再次查看)

## 常见问题

### Q1: 推送时提示 "repository not found"
**解决**: 先在 GitHub 创建仓库 https://github.com/new

### Q2: 推送时提示 "Permission denied"
**解决**: 使用 Token 而不是密码登录

### Q3: Vercel 部署失败
**解决**: 检查 vercel.json 文件是否存在

### Q4: 无法安装 Vercel CLI
**解决**: 使用网页版部署 https://vercel.com/new
